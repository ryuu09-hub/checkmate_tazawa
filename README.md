## PJ詳細
### セルフレジのプロトタイプの作成 
・カメラを接続したRaspberry Pi上で、リアルタイムに動作するプロトタイプを作成する

### 業務要件
・最低５種類の任意のペットボトル飲料に対して、商品名と価格を3秒以内に表示すること
　(エラー率１０％以内)

・追加で選定されたペットボトル飲料に対してはエラー率を２０％以内

・評価を行いどちらのチームにセルフレジの開発を依頼するかを決定する

### 使用モデル
Mobilenetv2
